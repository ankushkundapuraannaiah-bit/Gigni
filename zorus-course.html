Course Curriculum

12 Weeks of Real Python

From core fundamentals to production deployment — every week pushes you further.

Week 1 – 2

Python Core & Environment

Set up your dev environment and master Python syntax, data types, and control flow.

  • Variables, types & operators
  • Functions & scope
  • Lists, dicts & tuples
  • File I/O & exceptions
Week 3 – 4

OOP & Modules

Write structured, reusable code using object-oriented principles and Python's module system.

  • Classes & inheritance
  • Decorators & iterators
  • Python standard library
  • Virtual envs & pip
🔒 Unlocks Week 5
Week 5 – 6

Web APIs with FastAPI

Build and deploy production-grade REST APIs using FastAPI and Pydantic.

  • HTTP fundamentals
  • FastAPI routing & models
  • Authentication & JWT
  • API documentation
🔒 Unlocks Week 7
Week 7 – 8

Databases & SQL

Persist data reliably using PostgreSQL and SQLAlchemy ORM with proper migrations.

  • SQL fundamentals
  • SQLAlchemy ORM
  • Migrations with Alembic
  • Query optimization
🔒 Unlocks Week 9
Week 9 – 10

Automation & Data

Automate workflows and analyse data with Pandas, NumPy, and scripting pipelines.

  • Pandas & NumPy basics
  • Web scraping
  • Task automation
  • Data visualisation
🔒 Unlocks Week 11
Week 11 – 12

Deployment & Capstone

Ship your capstone project to production using Docker, GitHub Actions, and cloud hosting.

  • Docker & containers
  • CI/CD with GitHub Actions
  • Cloud deployment (Vercel/Render)
  • Capstone project review
