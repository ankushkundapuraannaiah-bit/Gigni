✦ Course Modules

Three months of real building.

Each module is anchored by a reference video and a concrete assignment. Watch, understand, then go build your own working version.

1
AI Model Development
Create a Realistic Talking AI Clone using HeyGen
Month 1 — Assignment Overview
Generative Identity Build

In this first module, you will apply the concepts from the reference video to build your own generative identity and agentic workflow. Record yourself, process through HeyGen, clone your voice, and create a working AI avatar that speaks as you.

Deliverables
  • Record a 2-minute baseline video with clear lighting and audio quality.
  • Process the footage through HeyGen to calibrate your personal avatar model.
  • Clone your voice profile and generate a 30-second introductory script.
  • Submit code, working demo video, and a README file for verification.
2
Automating Workflows with n8n
Build Visual AI Workflows & Automation Pipelines
Month 2 — Assignment Overview
N8N Agentic Automation

In this module, you'll learn to design visual AI workflows. You will set up n8n and build custom pipelines to integrate AI agents into real-world applications — connecting APIs, scheduling tasks, and orchestrating multi-step logic.

Deliverables
  • Initialize a basic n8n workspace (Cloud or Self-Hosted setup).
  • Construct a multi-node workflow linking an external API to an AI agent.
  • Create an automated scheduling or content pipeline that runs end-to-end.
  • Submit code, working demo video, and a README file for verification.
3
Turning Workflows into Web Apps
Build frontend interfaces for your n8n pipelines
Month 3 — Assignment Overview
Web App Integration

In this module, you will learn how to connect your n8n workflows directly to a frontend web application. Using webhook bridges, you'll turn automated background tasks into interactive, user-facing tools that anyone can use.

Deliverables
  • Configure Webhook nodes in n8n to receive and process user input.
  • Design and connect a frontend web interface to your workflow backend.
  • Deploy the full-stack web application live or record a working local demo.
  • Submit code, working demo video, and a README file for verification.
